One-time setup
An Apps Script syncs your inbox into a Google Sheet with AI classification. The sheet is published as CSV for this page to read. See inbox_script.js in the repo for the Apps Script code.
1
Set up an OpenAI API key and add the code from inbox_script.js to your Apps Script project.
2
Run syncInbox and set a time trigger (every 15 min).
3
In the Sheet: File → Share → Publish to web → select the inbox sheet → CSV → Publish. Copy the URL.
If you already set up the Canvas emails sheet, you can reuse the same spreadsheet — the script creates its own "inbox" tab. The Gemini API free tier handles this volume easily.
