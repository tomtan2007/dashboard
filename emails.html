Connect your Gmail
This page reads your Canvas notification emails directly from Gmail. You need a Google OAuth Client ID — it takes about 2 minutes to set up and only needs to be done once.
1
Go to console.cloud.google.com/apis/credentials and create a project (or select an existing one).
2
Click Enable APIs and enable the Gmail API.
3
Go to Credentials → Create Credentials → OAuth client ID. Choose Web application.
4
Under Authorized JavaScript origins, add
5
Copy the Client ID and paste it below.
