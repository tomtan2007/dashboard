One-time setup — about 5 minutes
This uses a Google Apps Script that syncs your Canvas emails into a Google Sheet on a timer. The sheet is then published as a public CSV that this page can read — no login redirects, no popups, no API keys.
1
Create a new Google Sheet in your UMich account. Name it anything (e.g. "Canvas Emails Data"). Copy the spreadsheet ID from the URL — it's the long string between /d/ and /edit.
2
Open script.google.com, click New project, and paste the code below. Replace YOUR_SHEET_ID with the ID you copied.
function syncEmails() { var threads = GmailApp.search( 'from:instructure.com OR from:gradescope.com OR from:perusall.com', 0, 100); var emails = []; for (var i = 0; i < threads.length; i++) { var msgs = threads[i].getMessages(); for (var j = 0; j < msgs.length; j++) { var m = msgs[j]; var body = m.getPlainBody().split('________')[0].trim(); emails.push([ m.getId(), m.getSubject(), m.getFrom(), m.getDate().getTime(), body.substring(0, 600), m.isUnread() ]); } } emails.sort(function(a, b) { return b[3] - a[3]; }); var ss = SpreadsheetApp.openById('YOUR_SHEET_ID'); var sheet = ss.getSheetByName('emails'); if (!sheet) sheet = ss.insertSheet('emails'); sheet.clearContents(); sheet.getRange(1, 1, 1, 6).setValues( [['id','subject','from','date','body','unread']] ); if (emails.length > 0) { sheet.getRange(2,1,emails.length,6).setValues(emails); } }
3
Run it once: Click the ▶ Run button in the toolbar. Google will ask you to authorize access to Gmail and Sheets — click through and allow it. You should see data appear in your spreadsheet.
4
Set up auto-sync: In the left sidebar click the clock icon (Triggers) → Add Trigger. Set function to syncEmails, event source to Time-driven, and pick an interval (e.g. every 15 minutes).
5
Publish the sheet: In your Google Sheet go to File → Share → Publish to web. Select the emails sheet and CSV format. Click Publish and copy the URL.
The published CSV URL is public and requires no login. It updates automatically whenever the script syncs. The URL is saved in your browser's localStorage.
